
# CatalogoApp

CatalogoApp es una aplicación web desarrollada en ASP.NET Core MVC que permite gestionar un catálogo de películas o videojuegos. La aplicación incluye funcionalidades como el registro e inicio de sesión de usuarios, la creación y visualización de elementos dentro del catálogo, así como la posibilidad de que los usuarios autenticados agreguen reseñas a cada elemento. Toda la información se almacena mediante archivos JSON, lo que permite manejar los datos sin necesidad de una base de datos. El proyecto está orientado al aprendizaje de la arquitectura MVC, el manejo de sesiones y la persistencia de datos en aplicaciones web.

---

## Tecnologías usadas

- ASP.NET Core MVC (.NET)
- C#
- HTML5
- CSS3
- Sesiones en ASP.NET Core
